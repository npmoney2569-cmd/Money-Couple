# โมดูลเสริม (Extended Modules)

## 1. เป้าหมายการออม

- ตั้งเป้าหมาย (ชื่อ, จำนวนเงินเป้า, วันครบกำหนด)
- ฝาก / ถอนเงินเข้าเป้าหมาย
- แสดงความคืบหน้า (Progress bar + %)
- แจ้งเตือนเมื่อถึงเป้า

## 2. หนี้สิน

- หนี้เรา (เราเป็นผู้กู้)
- หนี้คนอื่น (เราเป็นผู้ให้กู้)
- ดอกเบี้ย
- งวดชำระ
- บันทึกการชำระแต่ละงวด
- ตารางผ่อนชำระ (Amortization schedule)
- ชื่อคู่สัญญาและวันครบกำหนด
- แจ้งเตือนก่อนถึงวันชำระ

## 3. สินทรัพย์

ประเภท:

- บ้าน
- รถ
- หุ้น
- กองทุน
- Crypto
- เงินฝาก

ฟีเจอร์:

- ประวัติการอัปเดตมูลค่า (Valuation history)
- ค่าเสื่อมราคา (Depreciation) สำหรับรถ
- Portfolio tracking สำหรับหุ้น/กองทุน/Crypto (ผลตอบแทน, เงินปันผลแยกจากรายรับทั่วไป)

## 4. รายการประจำ

- รายรับ/รายจ่ายที่เกิดซ้ำ (เงินเดือน, ค่าเช่า)
- ตั้งความถี่ (รายวัน / สัปดาห์ / เดือน / ปี)
- สร้างรายการอัตโนมัติตามกำหนด (cron job)

## 5. บิล/Subscription

- วันครบกำหนดจ่ายบิล (ค่าน้ำ, ค่าไฟ, บัตรเครดิต)
- Subscription tracker (Netflix, Spotify ฯลฯ)
- แจ้งเตือนก่อนถึงวันหักเงิน (Push / Email / LINE reply)

## 6. รายงาน

### รายงานพื้นฐาน (MVP)

- รายวัน / รายเดือน / รายปี
- Cash Flow
- สรุปตามหมวดหมู่
- Export CSV

### รายงานขั้นสูง (Phase 2+)

- Budget vs Actual
- Net Worth ตามเวลา
- เปรียบเทียบช่วงเวลา (MoM, YoY)
- Custom report builder
- ตั้งเวลาส่งรายงานอัตโนมัติ (อีเมลสรุปทุกต้นเดือน)
- Export PDF / Excel / CSV

## 7. แจ้งเตือน

### ประเภทการแจ้งเตือน

- บิล/Subscription ใกล้ครบกำหนด
- ใช้เงินเกินงบที่ตั้งไว้
- เป้าหมายการออมถึงเป้า
- หนี้ใกล้ครบกำหนดชำระ

### ช่องทางแจ้งเตือน

| ช่องทาง | ค่าใช้จ่าย | Phase |
|---------|-----------|-------|
| Push Notification ในแอป | ฟรี | Phase 2 |
| Email | ตาม provider | Phase 2 |
| LINE Bot (reply ภายใน 24 ชม.) | ฟรี | Phase 3 |
