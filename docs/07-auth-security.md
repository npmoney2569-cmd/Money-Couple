# ความปลอดภัย & การยืนยันตัวตน

## Login / Register

รองรับ Login แบบผสม เลือกวิธีใดวิธีหนึ่งเพื่อสมัคร/เข้าระบบ:

| วิธี | Phase |
|------|-------|
| Email + Password | MVP |
| Google Login | MVP |
| Apple Login | Phase 2 |
| LINE Login | Phase 3 |

## ผูกช่องทางเพิ่มเติม (Account Linking)

- สมัครด้วยวิธีใดวิธีหนึ่งก่อน แล้วเข้าหน้าตั้งค่า → "บัญชีที่เชื่อมต่อ" เพื่อผูกช่องทางอื่น
  - สมัครด้วย Email → ผูก LINE ทีหลังเพื่อเปิดใช้ LINE Bot
  - สมัครด้วย LINE Login → ผูก Email ทีหลังเพื่อกู้คืนบัญชี
- ทุกช่องทางที่ผูกแล้ว Login เข้าบัญชีเดียวกันได้ (อ้างอิง `user_id` เดียวกัน)
- ป้องกันการผูกอีเมล/LINE ซ้ำ (unique constraint บน `provider` + `provider_uid`)
- ยกเลิกการผูกได้ ยกเว้นกรณีเหลือช่องทางเดียว (ต้องมีอย่างน้อย 1 ช่องทางเสมอ)

## ความปลอดภัยอื่นๆ

| ฟีเจอร์ | Phase |
|---------|-------|
| PIN Lock / Biometric Lock | Phase 2 |
| Two-Factor Authentication (2FA) | Phase 3 |
| Audit Log — ประวัติแก้ไข/ลบรายการ | Phase 2 |
| Permissions — สิทธิ์ดูอย่างเดียว/แก้ไขได้ (โหมดคู่รัก) | Phase 3 |
| จัดการคำเชิญความสัมพันธ์ (ยืนยัน/ปฏิเสธ/ยกเลิก) | Phase 3 |

## ตั้งค่าทั่วไป

- ภาษา / สกุลเงินหลัก / รูปแบบวันที่
- Theme (Light / Dark)
- Backup/Restore อัตโนมัติ (พร้อมเข้ารหัสไฟล์) — Phase 3
- การลบบัญชี/ข้อมูลถาวร
