# โมดูลหลัก (Core Modules)

## 1. Dashboard

แสดงภาพรวมการเงินในหน้าเดียว

- เงินคงเหลือ (รวมทุกบัญชี)
- รายรับ / รายจ่าย (ช่วงเวลาที่เลือก)
- เงินออม
- หนี้
- Net Worth
- กราฟ (รายรับ-รายจ่าย, แยกตามหมวดหมู่)
- รายการล่าสุด (5–10 รายการ)

## 2. รายรับ

### ข้อมูลที่บันทึก

- วันที่
- จำนวนเงิน
- หมวดหมู่
- บัญชี
- ผู้รับเงิน
- แท็ก
- โน้ต
- ไฟล์แนบ
- ช่องทางบันทึก (`app` / `line_bot` / `ocr`)

### หมวดหมู่เริ่มต้น

- เงินเดือน
- โบนัส
- ฟรีแลนซ์
- ดอกเบี้ย
- ปันผล
- อื่นๆ

## 3. รายจ่าย

### ข้อมูลที่บันทึก

- วันที่
- จำนวนเงิน
- หมวดหมู่
- บัญชี
- ร้านค้า
- ใบเสร็จ (รูปภาพ)
- แท็ก
- โน้ต
- ช่องทางบันทึก (`app` / `line_bot` / `ocr`)

### หมวดหมู่เริ่มต้น

- อาหาร
- เดินทาง
- บ้าน
- ค่าน้ำ
- ค่าไฟ
- อินเทอร์เน็ต
- สุขภาพ
- ท่องเที่ยว
- ช้อปปิ้ง
- ประกัน
- ภาษี

### Split Transaction

รายการเดียวแยกได้หลายหมวดหมู่ (เช่น บิลห้างที่มีทั้งอาหารและของใช้) — เก็บใน `transaction_splits`

## 4. โอนเงิน

โอนเงินระหว่างบัญชีของตัวเอง ไม่นับเป็นรายรับหรือรายจ่าย

### ข้อมูลที่บันทึก

- วันที่
- จำนวนเงิน
- บัญชีต้นทาง
- บัญชีปลายทาง
- ค่าธรรมเนียม (ถ้ามี)
- โน้ต

### พฤติกรรม

- หักยอดจากบัญชีต้นทาง บวกยอดที่บัญชีปลายทาง
- แสดงในรายการประวัติแยกประเภท `transfer`
- ไม่กระทบงบประมาณรายจ่าย (ยกเว้นค่าธรรมเนียม — นับเป็นรายจ่ายแยกได้)

## 5. บัญชี

ประเภทบัญชีที่รองรับ:

- เงินสด
- ธนาคาร
- บัตรเครดิต
- e-Wallet
- บัญชีลงทุน

### ข้อมูลที่บันทึก

- ชื่อบัญชี
- ประเภท
- ยอดคงเหลือเริ่มต้น
- สกุลเงิน
- สี / ไอคอน
- สถานะ (ใช้งาน / ปิด)

## 6. หมวดหมู่

- หมวดหมู่หลัก (Preset) — แยกรายรับ/รายจ่าย
- หมวดหมู่ย่อย (Subcategory) เช่น "อาหาร" → "ร้านอาหาร", "ของสด", "เดลิเวอรี่"
- หมวดหมู่ที่ผู้ใช้สร้างเอง (Custom) พร้อม icon/color
- จัดการแท็ก (Tag management) แยกจากตัวรายการ

## 7. ค้นหา/กรองข้อมูล

- ค้นหาตามช่วงวันที่
- กรองตามหมวดหมู่ / บัญชี / แท็ก / ช่วงจำนวนเงิน
- กรองตามช่องทางบันทึก (`app` / `line_bot` / `ocr`)
- กรองตามประเภท (รายรับ / รายจ่าย / โอนเงิน)

## 8. งบประมาณ

- รายวัน
- รายสัปดาห์
- รายเดือน
- รายปี
- Progress bar แสดง % ที่ใช้ไป
- แจ้งเตือนเมื่อใกล้หรือเกินงบ (Phase 2)

## 9. ปฏิทิน

แสดงรายการทางการเงินบนปฏิทินรายเดือน

### สิ่งที่แสดงบนปฏิทิน

- รายรับ / รายจ่าย (จุดสีตามประเภท)
- วันครบกำหนดบิล / Subscription
- วันชำระหนี้
- รายการประจำที่จะเกิดขึ้น

### การทำงาน

- คลิกวันที่ → ดูรายการทั้งหมดของวันนั้น
- ลากเพื่อเพิ่มรายการใหม่ (optional, Phase 2+)
- สลับมุมมอง เดือน / สัปดาห์

## 10. ตั้งค่า

### ทั่วไป

- ภาษา (ไทย / อังกฤษ)
- สกุลเงินหลัก
- รูปแบบวันที่
- Theme (Light / Dark / System)

### บัญชีที่เชื่อมต่อ

- ผูก/ยกเลิก Google, Apple, LINE, Email
- ดูรายละเอียดใน [07-auth-security](./07-auth-security.md)

### ข้อมูล

- Backup / Restore อัตโนมัติ (พร้อมเข้ารหัสไฟล์) — Phase 3
- Export ข้อมูลทั้งหมด (CSV/JSON) — MVP
- ลบบัญชี / ข้อมูลถาวร

### ความปลอดภัย

- PIN Lock / Biometric Lock — Phase 2
- Two-Factor Authentication (2FA) — Phase 3

## 11. ผู้ใช้งาน

### โปรไฟล์

- ชื่อที่แสดง
- รูปโปรไฟล์
- อีเมล (ถ้ามี)
- วันที่สมัคร

### การจัดการบัญชี

- แก้ไขโปรไฟล์
- เปลี่ยนรหัสผ่าน (กรณีสมัครด้วย Email)
- ดูช่องทาง Login ที่เชื่อมต่อ
- ลบบัญชี

### โหมดคู่รัก (Phase 3)

- ส่งคำเชิญคู่รัก
- ยืนยัน / ปฏิเสธ / ยกเลิกความสัมพันธ์
- กำหนดสิทธิ์ (ดูอย่างเดียว / แก้ไขได้)
